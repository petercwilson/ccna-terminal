# CCNA::TERMINAL

An interactive CCNA 200-301 study trainer with a retro hacker terminal aesthetic.

## Modules

- **Subnet Drill** — CIDR quizzes + Magic Number Calculator with live bit-map visualization
- **Topo Builder** — Drag-and-drop network topology builder with challenge validation
- **Route Select** — Routing table scenarios testing longest prefix match and Administrative Distance
- **VLAN/STP Drop** — Drag-and-drop exercises for VLAN assignment and STP port states

Each module includes a full **LEARN tab** with concept guides, cheat sheets, config examples, and exam tips.
